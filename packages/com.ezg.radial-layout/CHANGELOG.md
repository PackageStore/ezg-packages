# Changelog

## [0.1.0] - 2026-06-20
### Added
- Initial release extracted from `Assets/_Project/3rdParty/Radial Layout`.
- `RadialLayout_v2` LayoutGroup: arranges children around a circular arc with configurable radius (`fDistance`), angle range (`MinAngle`/`MaxAngle`/`StartAngle`), visible-only layout, and per-child rotation (`VectorRotation` face-center mode plus `ChildRotation` offset).
