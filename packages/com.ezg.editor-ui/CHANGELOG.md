# Changelog

### Added
- Initial release extracted from `Assets/_Project/Editor/Shared/EditorBaseStyle`.

- Drawer helpers for card headers, chips, search fields, panel backgrounds and rounded borders.
- Editor-only `Ezg.EditorUI` assembly with no runtime code and no third-party dependencies.
