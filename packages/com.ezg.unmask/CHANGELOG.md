# Changelog

## [0.1.0] - 2026-06-20
### Added
- Initial release extracted from `Assets/_Project/3rdParty/Unmask/`.
- `Unmask` component — reverse masking through a parent UI `Mask` with edge
  smoothing, fit-to-target (with optional per-frame `LateUpdate` fit), padding,
  offset, only-for-children mode, and optional image-property syncing.
- `UnmaskRaycastFilter` component — lets pointer input pass through the unmasked
  rectangle.
